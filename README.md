## Características principales

Gran parte de las funcionalidades que presenta la aplicacion están inspiradas de otras del mismo rubro como Evernote o SimpleNotes.

La aplicacion incluye:

- Creacion, edicion y gestión de Notas, listas de tareas, libretas, y agendas. 
- Interfaz sencilla
- Diferentes fuentes para la redacción de notas, así como colores y formatos.
- Insercción de tablas, gráficos e imágenes.
- Calendario.
- Alarmas y notificaciones. 
- Diferentes planes de subcripción. (Como la aplicación no está pensanda con fines comerciales, todos los planes tendran un mismo costo pero segun sea el selecionado, se habilitarás diferentes características)

## Desarrollo

BlueNotes es una SPA construida con la librería de React.js, JavaScript, Sass y los servicios de Firebase.
